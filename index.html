Einstellungen & Daten
Dieses Lernprogramm lädt Inhalte aus data/unit3.json.

Eigenes Unit‑3‑Material einfügen

  1. Lege die Fotos (Screenshots) in den Ordner assets/.
  2. Öffne data/unit3.json und trage deine Inhalte ein (Texte, Vokabeln, Aufgaben, Bildpfade).
  3. Starte neu / aktualisiere die Seite.
JSON‑Format (Kurzinfo) 
{
  "unit": "3",
  "title": "Unit 3",
  "topics": [
    {
      "id": "topic-1",
      "title": "Topic title",
      "tags": ["vocabulary","grammar"],
      "intro": "Kurzer Überblick…",
      "image": "assets/yourphoto1.jpg",
      "sections": [
        {
          "id":"sec-1",
          "title":"Unterkapitel",
          "theory":[{"h":"Merksatz","p":"…"}],
          "vocab":[{"term":"…","definition":"…","example":"…"}],
          "tasks":[
            {"type":"mcq","q":"…","choices":["A","B","C"],"answer":1,"explain":"…"},
            {"type":"fill","q":"I ____ to school.","answer":"go","hint":"Verb: go"},
            {"type":"match","pairs":[["term","definition"],["…","…"]]}
          ]
        }
      ]
    }
  ]
}
Hinweis: Lokal geladene JSON‑Datei wird nur für diese Sitzung genutzt (nicht gespeichert).

Bedienung

  • Lernen: Inhalte lesen, Fotos ansehen, Unterkapitel markieren.
  • Karten: Vokabeln als Lernkarten (spaced repetition light).
  • Üben: Gemischte Aufgaben (MC, Lücke, Zuordnen).
  • Quiz: Kapitel‑Tests & Unit‑Test.
  • Glossar: Gesamte Wortliste mit Suche.
